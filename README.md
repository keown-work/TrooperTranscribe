# KSPTranscribe
A USB-deployable, browser-based transcription tool built for Kentucky State Police investigators. Supports multi-speaker diarization, bulk speaker reassignment, and export to DOCX, PDF, TXT, SRT, VTT, and CSV. Powered by faster-whisper and pyannote.audio. Fully air-gappable, no external server contact. Optimized for standard-issue Latitude MDT.
